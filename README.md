# 프로젝트 개발환경
 - 운영체제 : Window
 - 통합개발환경 : STS4
 - JDK 버전 : JDK 11
 - 데이터베이스 : MySQL
 - 빌드 툴 : Maven
 - 관리 툴 : Git

# 프로젝트 기술 스택
 - 프론트엔드 : HTML, CSS, JS, JQuery, AJax
 - 백엔드 : Spring Boot, Java, Jsp
 - 데이터베이스 : MySQL, Apache Tomcat, MyBatis
